== Description ==

The Robi Pro Preloader wordPress plugin.It will be enable a Preloader on your web site. You can change the preloader image/design from settings.

<h3>Features:</h3>

<p>After install, the plugin will add a menu under Settings called "Robi Pro  Preloader Settings". </p>

== Installation ==

1. Upload `robi-pro-preloader` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings menus change option what you want.
